# MayaGaussianSplatting

remember to set 

`DEVKIT_LOCATION` - user customized path for maya plugin, we will copy .mll to this dir in post-build stage

`MAYA_LOCATION` - maya installment dir for lib&include search

this two variables in environment variables setting



This project use DX11 as the render API,

please set here:

Windows → Settings/Preferences → Preferences
  → Display → Viewport 2.0
  → Rendering Engine: DirectX 11
